## Chạy dự án

```bash
pip install -r requirements.txt
python main.py
```

## Cấu trúc ngắn

- `main.py` - entry point
- `config.py` - cấu hình
- `utils.py` - helper chung
- `cogs/` - commands
- `services/` - business logic
- `models/` - data structures
- `docs/` - tài liệu
