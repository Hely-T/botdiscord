# Music Player Assets

Các file tùy chọn:

- `player_background.png`: nền mặc định của canvas player.
- `player_frame.png`: khung phủ lên toàn bộ canvas, kích thước khuyến nghị `1000x360`.

Nếu không có file, player tự tạo nền từ thumbnail video. Có thể đổi nền trực tiếp
trên Discord bằng `play settings background <url>` hoặc gửi ảnh đính kèm.
